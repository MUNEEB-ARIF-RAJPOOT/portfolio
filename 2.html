My Projects

QuoteGenie

QuoteGenie is a motivational quote generator that displays a random quote each time the user clicks a button. It’s a fun and inspiring tool for users who need a quick boost of positivity or creativity. The quotes are stored in a simple JavaScript array.

ToDoMate

ToDoMate is a simple to-do list app where users can add tasks, mark them as complete, and delete them. It helps users stay organized by managing their daily tasks in one place. The app uses browser local storage to save tasks even after the page is refreshed.

SimpleCalc

SimpleCalc is a basic calculator web application that allows users to perform arithmetic operations like addition, subtraction, multiplication, and division. It’s designed to help users quickly solve math problems without needing a physical calculator.
